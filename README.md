# DungeonQuestTracker

A TBC Classic addon that tracks dungeon quest completion across all Classic and TBC dungeons.

## Features

- Track quest completion status for all Classic and TBC dungeon quests
- Quests grouped by dungeon complex with completion counts
- Three quest states: Completed (green), In Progress (yellow), Not Started (red)
- Prerequisite quest chains shown indented below parent quests
- Heroic-only quest tags
- Faction-aware filtering (Alliance/Horde)
- Click any quest to get a copyable Wowhead link
- Summary tab with overall progress overview
- Minimap button and keybinding support

## Slash Commands

- `/dqt` - Toggle quest tracker window
- `/dqt config` - Open options
- `/dqt summary` - Print completion summary to chat

## Dungeons Tracked

### Classic
Ragefire Chasm, Wailing Caverns, The Deadmines, Shadowfang Keep, Blackfathom Deeps, The Stockade, Gnomeregan, Scarlet Monastery (all wings), Razorfen Kraul, Razorfen Downs, Uldaman, Zul'Farrak, Maraudon, Sunken Temple, Blackrock Depths, Lower/Upper Blackrock Spire, Dire Maul (all wings), Stratholme, Scholomance

### TBC
Hellfire Ramparts, Blood Furnace, Shattered Halls, Slave Pens, Underbog, Steamvault, Mana-Tombs, Auchenai Crypts, Sethekk Halls, Shadow Labyrinth, Mechanar, Botanica, Arcatraz, Old Hillsbrad, Black Morass, Magisters' Terrace

## Installation

1. Download from CurseForge or clone this repository
2. Place the `DungeonQuestTracker` folder in your `Interface/AddOns` directory
3. Restart WoW or `/reload`
