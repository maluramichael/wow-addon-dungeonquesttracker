# DungeonQuestTracker

Never miss a dungeon quest again. DungeonQuestTracker shows you every quest available in every Classic and TBC dungeon, tracks your completion progress, and highlights prerequisite chains so you know exactly what to pick up before zoning in.

## Features

- Tracks 160+ dungeon quests across all Classic and TBC dungeons
- Quests grouped by dungeon with completion counts (completed/total)
- Three quest states: Completed (green checkmark), In Progress (yellow), Not Started (red X)
- Prerequisite quest chains displayed with follow-up quests indented below
- Quest objective summaries shown in tooltips on hover
- Heroic-only quest tags
- Faction-aware filtering (shows only your faction's quests)
- Click any quest to get a copyable Wowhead link
- Summary tab with overall progress breakdown by dungeon complex
- Minimap button with progress tooltip
- Keybinding support
- Configurable: hide completed dungeons, toggle prerequisites, heroic quests, faction filter

## Slash Commands

- `/dqt` - Toggle quest tracker window
- `/dqt config` - Open options
- `/dqt summary` - Print completion summary to chat

## Installation

1. Download from CurseForge or clone this repository
2. Place the `DungeonQuestTracker` folder in your `Interface/AddOns` directory
3. Restart WoW or `/reload`
